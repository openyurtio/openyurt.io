зүҲжң¬пјҡv1.2

иҫ№зјҳIngress

жң¬ж–ҮжЎЈд»Ӣз»ҚеҰӮдҪ•еңЁдә‘иҫ№еҚҸеҗҢеңәжҷҜдёӢйҖҡиҝҮиҫ№зјҳIngressи®ҝй—®жҢҮе®ҡиҠӮзӮ№жұ жҸҗдҫӣзҡ„жңҚеҠЎгҖӮ

е…·дҪ“еә”з”ЁеңәжҷҜдёәпјҡ

  1. иҠӮзӮ№жұ еҶ…жҲ–иҠӮзӮ№жұ еӨ–йҖҡиҝҮиҫ№зјҳingressи®ҝй—®иҠӮзӮ№жұ еҶ…жҸҗдҫӣзҡ„жңҚеҠЎгҖӮ
  2. иҠӮзӮ№жұ еӨ–и®ҝй—®nginx ingress controllerпјҢзӣ®еүҚж”ҜжҢҒйҖҡиҝҮNodePort ServiceеҸҠexternalIPsзҡ„ж–№ејҸгҖӮ

е…·дҪ“з”Ёжі•дёәпјҡ

  1. еҗҜз”ЁжҢҮе®ҡиҠӮзӮ№жұ дёҠзҡ„иҫ№зјҳIngressеҠҹиғҪгҖӮ
  2. еҗҢK8SдёҖж ·еҲӣе»әе№¶йғЁзҪІingressи§„еҲҷд»Ҙи®ҝй—®зӣёеә”зҡ„жңҚеҠЎгҖӮ

иҜ·жҢүд»ҘдёӢжӯҘйӘӨе°қиҜ•дҪҝз”Ёиҫ№зјҳIngressеҠҹиғҪпјҡ

1.еҗҜз”ЁжҢҮе®ҡиҠӮзӮ№жұ дёҠзҡ„иҫ№зјҳIngressеҠҹиғҪвҖӢ

YurtIngress opeatorиҙҹиҙЈе°Ҷnginx ingress controllerзј–жҺ’еҲ°йңҖиҰҒеҗҜз”Ёиҫ№зјҳIngressеҠҹиғҪзҡ„иҠӮзӮ№жұ дёӯгҖӮ еҒҮи®ҫжӮЁзҡ„OpenYurtйӣҶзҫӨдёӯжңү4дёӘиҠӮзӮ№жұ пјҡpool01гҖҒpool02гҖҒpool03гҖҒpool04пјҢеҰӮжһңжӮЁжғіиҰҒеңЁpool01е’Ңpool03дёҠеҗҜз”Ёиҫ№зјҳingressеҠҹиғҪпјҢеҸҜд»ҘжҢүеҰӮдёӢж–№ејҸеҲӣе»әYurtIngress CRпјҡ

1). еҲӣе»әYurtIngress CR yamlж–Үд»¶:

1.1пјү. YurtIngress CRзҡ„з®ҖеҚ•е®ҡд№ү:

apiVersion: apps.openyurt.io/v1alpha1 kind: YurtIngress metadata: name: yurtingress-test spec: pools:

  • name: pool01
  • name: pool03

й»ҳи®ӨдёәжҜҸдёӘиҠӮзӮ№жұ еҲӣе»әзҡ„nginx ingressжҺ§еҲ¶еҷЁеүҜжң¬ж•°дёә1
й»ҳи®Өзҡ„ingressжҺ§еҲ¶еҷЁdocker imageдёәпјҡk8s.gcr.io/ingress-nginx/controller:v0.48.1
й»ҳи®Өзҡ„з”ҹжҲҗingressжҺ§еҲ¶еҷЁwebhookиҜҒд№Ұзҡ„docker imageдёәпјҡk8s.gcr.io/ingress-nginx/kube-webhook-certgen:v0.48.1

1.2пјү. еҰӮжһңз”ЁжҲ·дёҚжғідҪҝз”Ёй»ҳи®Өзҡ„й…ҚзҪ®пјҢиҖҢжҳҜжғіеҜ№иҠӮзӮ№жұ еҒҡдёҖдәӣдёӘжҖ§еҢ–й…ҚзҪ®пјҢеҸҜд»ҘеҰӮдёӢе®ҡд№үCR:

apiVersion: apps.openyurt.io/v1alpha1 kind: YurtIngress metadata: name: yurtingress-test spec: ingress_controller_replicas_per_pool: 2 ingress_controller_image: k8s.gcr.io/ingress-nginx/controller:v0.49.0 ingress_webhook_certgen_image: k8s.gcr.io/ingress-nginx/kube-webhook-certgen:v0.49.0 pools:

  • name: pool01 ingress_ips:
    • xxx.xxx.xxx.xxx
  • name: pool03

е…¶дёӯпјҡ
igress_controller_replicas_per_pool/ingress_controller_image/ingress_webhook_certgen_imageеҸҜдҫӣз”ЁжҲ·иҮӘе®ҡд№үзӣёе…ій»ҳи®Өй…ҚзҪ®пјҢ ingress_ips д»ЈиЎЁеҰӮжһңз”ЁжҲ·жғійҖҡиҝҮexternalIPsзҡ„ж–№ејҸдёәжҹҗдёӘзү№е®ҡзҡ„иҠӮзӮ№жұ еҜ№еӨ–жҡҙйңІnginx ingressжҺ§еҲ¶еҷЁжңҚеҠЎзҡ„е…¬зҪ‘IPең°еқҖгҖӮ

жҸҗзӨәпјҡ

a). з”ЁжҲ·еҸҜд»ҘйҖҡиҝҮе®ҡд№үдёҚеҗҢзҡ„YurtIngress CRsжқҘеҜ№дёҚеҗҢиҠӮзӮ№жұ еҒҡдёҖдәӣдёӘжҖ§еҢ–й…ҚзҪ®пјҢжҜ”еҰӮйҖҡиҝҮе®ҡд№үдёҚеҗҢзҡ„CRжқҘеҜ№дёҚеҗҢзҡ„иҠӮзӮ№жұ й…ҚзҪ®дёҚеҗҢзҡ„ingressжҺ§еҲ¶еҷЁеүҜжң¬ж•°гҖӮ

b). еңЁspecдёӯпјҢвҖңingress_controller_replicas_per_poolвҖқиЎЁзӨәйғЁзҪІеңЁжҜҸдёӘиҠӮзӮ№жұ дёҠзҡ„ingressжҺ§еҲ¶еҷЁеүҜжң¬ж•°пјҢе®ғдё»иҰҒз”ЁдәҺHAй«ҳеҸҜз”ЁеңәжҷҜгҖӮ

c). еңЁspecдёӯпјҢвҖңpoolsвҖқиЎЁзӨәиҰҒеңЁе…¶дёҠејҖеҗҜingressеҠҹиғҪзҡ„иҠӮзӮ№жұ еҲ—иЎЁпјҢзӣ®еүҚж”ҜжҢҒиҠӮзӮ№жұ еҗҚеҸҠй’ҲеҜ№иҜҘиҠӮзӮ№жұ зҡ„ingressжңҚеҠЎе…¬зҪ‘IPй…ҚзҪ®гҖӮ

2). йғЁзҪІYurtIngress CR yamlж–Үд»¶пјҡ
еҒҮе®ҡCRж–Үд»¶еҗҚдёәyurtingress-test.yamlпјҡ

#kubectl apply -f yurtingress-test.yaml yurtingress.apps.openyurt.io/yurtingress-test created

з„¶еҗҺжӮЁеҸҜд»ҘжҹҘзңӢYurtIngress CRзҡ„зҠ¶жҖҒ:

    #kubectl get ying
    NAME                 REPLICAS-PER-POOL   READYNUM   NOTREADYNUM   AGE
    yurtingress-test     1                   2          0             3m13s

жҲҗеҠҹзј–жҺ’ingress controllerеҗҺпјҢжҜҸдёӘиҠӮзӮ№жұ е°ҶжҡҙйңІдёҖдёӘNodePortзұ»еһӢзҡ„ServiceжңҚеҠЎпјҡ

    #kubectl get svc -n ingress-nginx
    ingress-nginx  pool01-ingress-nginx-controller  NodePort  192.167.107.123  <none>   80:32255/TCP,443:32275/TCP  53m
    ingress-nginx  pool03-ingress-nginx-controller  NodePort  192.167.48.114   <none>   80:30531/TCP,443:30916/TCP  53m

жҸҗзӨә:

a). вҖңyingвҖқжҳҜYurtIngressиө„жәҗзҡ„з®Җз§°

b). еҪ“вҖңREADYNUMвҖқдёҺжӮЁйғЁзҪІзҡ„иҠӮзӮ№жұ ж•°йҮҸдёҖиҮҙж—¶пјҢиЎЁзӨәingressеҠҹиғҪеңЁжӮЁе®ҡд№үзҡ„жүҖжңүиҠӮзӮ№жұ дёҠе·Іе°ұз»ӘгҖӮ

c). еҪ“вҖңNOTREADYNUMвҖқдёҖзӣҙдёҚдёә0ж—¶пјҢеҸҜд»ҘжҹҘзңӢCRзҡ„зҠ¶жҖҒдәҶи§Јзӣёе…ідҝЎжҒҜпјҢжӮЁиҝҳеҸҜд»ҘжҹҘзңӢзӣёеә”зҡ„deploymentеҸҠpodд»ҘиҺ·еҸ–жӣҙиҜҰз»Ҷзҡ„й”ҷиҜҜдҝЎжҒҜпјҢд»ҺиҖҢжүҫеҮәingressеҠҹиғҪе°ҡжңӘе°ұз»Әзҡ„еҺҹеӣ гҖӮ

d). еҜ№дәҺжҲҗеҠҹеҗҜз”ЁingressеҠҹиғҪзҡ„жҜҸдёӘNodePoolпјҢдјҡдёәз”ЁжҲ·жҡҙйңІдёҖдёӘNodePortзұ»еһӢзҡ„жңҚеҠЎз”ЁжқҘи®ҝй—®nginx ingress controllerгҖӮ

e). YurtIngress operatorдјҡеҲӣе»әдёҖдёӘ"ingress-nginx"зҡ„namespaceпјҢзј–жҺ’nginx ingress controllerж—¶пјҢжүҖжңүи·ҹnamespaceзӣёе…ізҡ„resourceйғҪдјҡиў«йғЁзҪІеңЁиҝҷдёӘnamespaceдёӢгҖӮ

2.еҗҢK8SдёҖж ·еҲӣе»әе№¶йғЁзҪІingressи§„еҲҷд»Ҙи®ҝй—®зӣёеә”зҡ„жңҚеҠЎвҖӢ

еҪ“дёҠиҝ°жӯҘйӘӨ1е®ҢжҲҗеҗҺпјҢжӮЁе·Із»ҸйҖҡиҝҮYurtingressжҲҗеҠҹзҡ„е°Ҷnginx ingress controllerйғЁзҪІеҲ°зӣёеә”зҡ„иҠӮзӮ№жұ дёӯгҖӮжҺҘдёӢжқҘзҡ„з”Ёжі•е°ұе’ҢK8SдёӯдҪҝз”Ёingressзҡ„дҪ“йӘҢдёҖиҮҙдәҶгҖӮ

еҒҮи®ҫжӮЁзҡ„дёҡеҠЎеә”з”Ёиў«йғЁзҪІеҲ°дәҶеӨҡдёӘиҠӮзӮ№жұ дёӯпјҢе№¶дё”е®ғд»¬йҖҡиҝҮдёҖдёӘе…ЁеұҖзҡ„serviceеҜ№еӨ–жҡҙйңІпјҢдёҫдёӘдҫӢеӯҗпјҡ

apiVersion: apps/v1 kind: Deployment metadata: name: pool01-deployment labels: app: echo spec: replicas: 2 selector: matchLabels: app: echo template: metadata: labels: app: echo spec: containers:

  • name: echo-app image: hashicorp/http-echo args:
    • "-text=echo from nodepool pool01" imagePullPolicy: IfNotPresent nodeSelector: apps.openyurt.io/nodepool: pool01

apiVersion: apps/v1 kind: Deployment metadata: name: pool03-deployment labels: app: echo spec: replicas: 2 selector: matchLabels: app: echo template: metadata: labels: app: echo spec: containers:

  • name: echo-app image: hashicorp/http-echo args:
    • "-text=echo from nodepool pool03" imagePullPolicy: IfNotPresent nodeSelector: apps.openyurt.io/nodepool: pool03

kind: Service apiVersion: v1 metadata: name: echo-service spec: selector: app: echo ports:

  • port: 5678

еҪ“жӮЁжғіиҰҒи®ҝй—®pool01жҸҗдҫӣзҡ„жңҚеҠЎж—¶пјҢжӮЁеҸҜд»ҘеҰӮдёӢж“ҚдҪңпјҡ

1). еҲӣе»әingressи§„еҲҷyamlж–Үд»¶:

apiVersion: extensions/v1beta1 kind: Ingress metadata: name: ingress-pool01 annotations: kubernetes.io/ingress.class: pool01 ingress.kubernetes.io/rewrite-target: / spec: rules:

  • http: paths:
    • path: /echo backend: serviceName: echo-service servicePort: 5678

жҸҗзӨә:

a). з”ұе“ӘдёӘиҠӮзӮ№жұ жҸҗдҫӣingressеҠҹиғҪжҳҜз”ұingress classеҶіе®ҡзҡ„пјҢеӣ жӯӨжӮЁйңҖиҰҒе°Ҷingress classе®ҡд№үдёәжӮЁжғіиҰҒи®ҝй—®жңҚеҠЎзҡ„иҠӮзӮ№жұ еҗҚз§°гҖӮ

b). дёҚеҗҢK8SзүҲжң¬зҡ„ingress CRе®ҡд№үеҸҜиғҪдёҚеҗҢпјҢжӮЁйңҖиҰҒзЎ®дҝқingress CRзҡ„е®ҡд№үдёҺйӣҶзҫӨK8SзүҲжң¬еҢ№й…ҚгҖӮ

2). йғЁзҪІingressи§„еҲҷyamlж–Үд»¶:
еҒҮе®ҡyamlж–Үд»¶еҗҚдёәingress-myapp.yamlпјҡ

#kubectl apply -f ingress-myapp.yaml ingress.extensions/ingress-myapp created

жҲҗеҠҹе®ҢжҲҗдёҠиҝ°жүҖжңүжӯҘйӘӨеҗҺпјҢжӮЁе°ұеҸҜд»ҘйҖҡиҝҮingress controller NodePort serviceйӘҢиҜҒиҫ№зјҳIngressеҠҹиғҪдәҶпјҡ

#curl xxx:32255/echo

"xxx" д»ЈиЎЁиҠӮзӮ№жұ pool01дёӯзҡ„иҠӮзӮ№IPең°еқҖ "32255" д»ЈиЎЁеҜ№еә”иҠӮзӮ№жұ дёӯingress controllerжҡҙйңІзҡ„service NodePort

иҝ”еӣһз»“жһңеә”иҜҘдёҖзӣҙдёәпјҡ вҖңecho from nodepool pool01вҖқгҖӮ
